# Recruit-List
산업기능요원 채용 공고 정리 사이트

## 시스템

1. 병무청에서 병역 지정업체 검색

2. 해당 업체에 대한 채용공고를 긁어오는 방법이 정의되어 있으면 정의된 방법으로 채용공고 업데이트

3. 방법이 정의되어 있지 않다면 개발자가 채용공고에 대한 방법 정의 후 업데이트

### 스케줄링 주기

  매일 자정

## 기술스택

  Server
    
    * AWS EC2

    * MongoDB 4.0.16

    * MariaDB 10.4

  BackEnd

    * Java 1.8

    * Spring 5.2.3

    * Spring-Security 5.2.1

    * Junit 5

    * JPA, Hibernate

  FrontEnd

    React
